Flex

Una forma fácil y rápida de poder alinear y distribuir el espacio entre los elementos de un contenedor.

Para empezar a utilizar flex, necesitamos usar por defecto la clase flex en nuestro elemento.

Flex 1
Flex 2
Flex 1
Flex 2

<div class="flex mb-md">
    <div class="item">Flex 1</div>
    <div class="item">Flex 2</div>
</div>

<div class="flex flex-inline">
    <div class="item">Flex 1</div>
    <div class="item">Flex 2</div>
</div>

Dirección

Indica la dirección que van a seguir los elementos (por defecto es horizontal, por lo tanto no existe tal clase).

Como contabamos, no existe ninguna clase para indicar que funcione horizontalmente ya que por defecto el navegador funciona así. Aun así, podemos ordenar los elementos horizontalmente pero empezando por el lado contrario usando la clase row-reverse

Flex 1
Flex 2
Flex 3

<div class="flex row-reverse">
    <div class="item">Flex 1</div>
    <div class="item">Flex 2</div>
    <div class="item">Flex 3</div>
</div>

No solo podemos alinear los elementos horizontalmente, sino que también podemos hacerlo verticalmente con la clase column y verticalmente empezando por el lado opuesto usando column-reverse.

Flex grow y flex shrink

Usando la clase flex-item conseguimos que el elemento rellene todo el espacio que pueda del contenedor, mientras que los demas elementos utilizaran sólo el espacio que necesitan.

    <div class="item">Flex item</div>
    <div class="item flex-item">Flex item</div>
    <div class="item">Flex item</div>
    <div class="item">Flex item</div>
</div>
