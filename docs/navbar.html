Navbar

Barra de navegación responsive.

Para empezar a usar el componente navbar, debemos crear un contenedor que englobe a todo lo que vamos a escribir con la clase navbar.


<nav class="navbar">
    ...
</nav>

El segundo paso, será añadir un nuevo contenedor llamado navbar-content.


<nav class="navbar">
    <div class="navbar-content">
        ...
    </div>
</nav>

Dentro de navbar-content, encontramos las clases navbar-left, navbar-right y navbar-center. Estas clases nos permitirán situar el contenido en la posición del contenedor que queramos.

.navbar-left
.navbar-center
.navbar-right 

<nav class="navbar">
    <div class="navbar-content">
        <div class="navbar-left">
            .navbar-left
        </div>
        <div class="navbar-center">
            .navbar-center
        </div>
        <div class="navbar-right">
            .navbar-right
        </div>
    </div>
</nav>

En nuestro navbar, lo más probable es que se encuentre el logo de nuestra página web. Por lo tanto, podemos usar la clase navbar-logo la cual hará que nuestra imagen funcione como un elemento de bloque.

.navbar-center
.navbar-right 

<nav class="navbar">
    <div class="navbar-content">
        <div class="navbar-left">
            <div class="navbar-logo">
                <img src="//ximg.es/150x40/dedede/aaa&text=LOGO" alt="">
            </div>
        </div>
        <div class="navbar-center">
            .navbar-center
        </div>
        <div class="navbar-right">
            .navbar-right
        </div>
    </div>
</nav>

Lo mismo sucede con los botones. Si contamos con un botón en el navbar, podemos añadir a un contenedor con la clase navbar-button.

            </div>
        </div>
    </div>
</nav>

Si contamos con enlaces, usaremos la clase navbar-link.

.navbar-center
.navbar-link
.navbar-link

    <div class="navbar-content">

        </div>
        <div class="navbar-right">
            <div class="navbar-link"><a href="">.navbar-link</a></div>
            <div class="navbar-link"><a href="">.navbar-link</a></div>
        </div>
    </div>
</nav>

Dentro del navbar, podemos añadir un elemento dropdown para que nos depsliegue un menú. Dicho elemento dropdown se aplica creando un contenedor la clase navbar-dropdown que englobe tanto el elemento que desplegará el menú como el menú en si.

El menú desplegable utilizará la clase navbar-dropdown-menu y cada elemento dentro de él llevará la clase navbar-dropdown-item.

Por último, el elemento que accione este menú debe ser un enlace a que lleve la clase navbar-dropdown-toggle y onclick="document.querySelector('.navbar-dropdown').classList.toggle('open')".

.navbar-link
.navbar-link
menú
Dropdown item 1
Dropdown item 2
Dropdown item 3

<nav class="navbar">
    <div class="navbar-content">
        <div class="navbar-left">
            <div class="navbar-logo">
                <img src="//ximg.es/150x40/dedede/aaa&text=LOGO" alt="">
            </div>
        </div>
        <div class="navbar-center">
            <div class="navbar-link"><a href="">.navbar-link</a></div>
            <div class="navbar-link"><a href="">.navbar-link</a></div>
        </div>
        <div class="navbar-right">
            <div class="navbar-dropdown">
                <a onclick="document.querySelector('.navbar-dropdown').classList.toggle('open')" class="navbar-dropdown-toogle">menú<i class="icon-chevron-down"></i></a>
                <div class="navbar-dropdown-menu">
                    <div class="navbar-dropdown-item"><a href="">Dropdown item 1</a></div>
                    <div class="navbar-dropdown-item"><a href="">Dropdown item 2</a></div>
                    <div class="navbar-dropdown-item"><a href="">Dropdown item 3</a></div>
                </div>
            </div>
        </div>
    </div>
</nav>

Para terminar, encontramos la clase navbar-responsive. En el contenedor que apliquemos esta clase volveremos a usar las mismas clases usadas anteriormente en navbar-content, es decir, navbar-left, navbar-right y navbar-center para estructurar nuestro menú responsive.

Dentro navbar-responsive encontramos navbar-burger conteniendo tres elementos con la clase navbar-burger-bar, los cuales representan el icono del menú y sus líneas. Esto se realiza así y no mediante un icono por el simple hecho de que así podemos animar el menú desplegable cuando este se abra y cierre.

.navbar-link
.navbar-item
Item 3
Dropdown item 1
Dropdown item 2
Dropdown item 3

<nav class="navbar">
    <div class="navbar-content">
        <div class="navbar-left">
            <div class="navbar-logo hidden-xs hidden-sm">
                <img src="//ximg.es/150x40/dedede/aaa&text=LOGO" alt="">
            </div>
        </div>
        <div class="navbar-center">
            <div class="navbar-link"><a href="javascript: void(0)">.navbar-link</a></div>
            <div class="navbar-item">.navbar-item</div>
        </div>
