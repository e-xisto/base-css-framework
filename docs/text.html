Display

Si necesitamos resaltar un título con mayor enfásis que un h1 utilizamos la clase display.

Párrafo como display


<p class="display">Párrafo como display</p>

Máyusculas y minúsculas

Para modificar el texto a minúsculas usaríamos la clase .text-lowercase, mientras que si lo deseamos en mayúsculas usariamos text-uppercase.

TEXTO MINÚSCULA

texto mayúsculas


<p class="text-lowercase">TEXTO MINÚSCULA</p>
<p class="text-uppercase">texto mayúsculas</p>

Nowrap

Para que el texto no salte de línea, tenemos la clase text-nowrap

.text-nowrap


<p class="text-nowrap">.text-nowrap</p>

Subrayado

Subraya un elemento usando text-decoration-underline o, por el contrario, eliminar el subrayado usando text-decoration-none

.text-decoration-underline

.text-decoration-none


<p class="text-decoration-underline">.text-decoration-underline</p>
<p class="text-decoration-none">.text-decoration-none</p>
