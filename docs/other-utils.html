Texto ellipsis

Para que nuestro texto muestre una ellipsis (...), añadiremos a dicho elemento que contiene el texto la clase ellipsis seguido del atributo data-ellipsis o data-ellipsis-height.

Si usamos data-ellipsis, indicaremos cuantas líneas queremos que ocupe el texto, mientras que si usamos data-ellipsis-height, definiremos el alto máximo que ocupará.

Clearfix

Borra fácilmente los floats de un elemento padre usando la clase clearfix.

.float-right

.float-left

no float

<div class="clearfix">
    <p class="float-right">.float-right</p>
    <p class="float-left">.float-left</p>
</div>
<p>no float</p>
