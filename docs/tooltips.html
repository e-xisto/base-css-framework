Tooltips

Basado en wenk https://github.com/tiaanduplessis/wenk.

Usa tooltips aplicando data-tooltip-*="".

Los data-* disponibles son los siguientes:

  • data-tooltip: el cual mostará el mensaje que queramos
  • data-tooltip-pos: para asignar la posición en la que se mostrará el mensaje.
  • data-tooltip-theme: para cambiar el estilo del contenedor del mensaje.
  • data-tooltip-length: para especificar el tamaño del mensaje.

Data-tooltip-pos

A data-tooltip-pos se le pueden aplicar los valores: top, bottom, left y right.

Ejemplo tooltip top

Ejemplo tooltip bottom

Ejemplo tooltip left

Ejemplo tooltip right


<p><span data-tooltip="This is a tooltip top!" data-tooltip-pos="top">Ejemplo tooltip top</span></p>
<p><span data-tooltip="This is a tooltip bottom!" data-tooltip-pos="bottom">Ejemplo tooltip bottom</span></p>
<p><span data-tooltip="This is a tooltip left!" data-tooltip-pos="left">Ejemplo tooltip left</span></p>
<p><span data-tooltip="This is a tooltip right!" data-tooltip-pos="right">Ejemplo tooltip right</span></p>

Data-tooltip-length

A data-tooltip-length podemos aplicarle los valores: small, medium, large y fit.

Ejemplo tooltip tamaño pequeño

Ejemplo tooltip tamaño medio

Ejemplo tooltip tamaño grande

Ejemplo tooltip tamaño ajustado al elemento


<p><span data-tooltip="Lorem ipsum dolor sit amet consectetur adipisicing elit." data-tooltip-length="small">Ejemplo tooltip tamaño pequeño</span><p>
<p><span data-tooltip="Lorem ipsum dolor sit amet consectetur adipisicing elit." data-tooltip-length="medium">Ejemplo tooltip tamaño medio</span><p>
<p><span data-tooltip="Lorem ipsum dolor sit amet consectetur adipisicing elit." data-tooltip-length="large">Ejemplo tooltip tamaño grande</span><p>
<p><span data-tooltip="Lorem ipsum dolor sit amet consectetur adipisicing elit." data-tooltip-length="fit">Ejemplo tooltip tamaño ajustado al elemento</span><p>

Data-tooltip-theme

Podemos aplicarle dark, que sería el valor por defecto, y light.

Ejemplo tooltip theme light

Ejemplo tooltip theme dark


<p><span data-tooltip="This is a tooltip!" data-tooltip-theme="light" data-tooltip-pos="bottom">Ejemplo tooltip theme light</span><p>
<p><span data-tooltip="This is a tooltip!" data-tooltip-theme="dark" data-tooltip-pos="top">Ejemplo tooltip theme dark</span><p>
