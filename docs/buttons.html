Botones

Para empezar a aplicar los estilos de los botones predefinidos por Base CSS, necesitamos usar la clase btn o button por defecto.

Esta clase se puede aplicar tanto a elementos <button> como <a>.

Para modificar el color de los botones, deberíamos cambiar los colores en el archivo de variables.

Default Primary
Danger Warning 

<a href="" class="btn">Default</a>
<a href="" class="btn btn-primary">Primary</a>

<button class="btn btn-secondary">Secondary</button>
<button class="btn btn-link">link</button>

<a href="" class="btn btn-danger">Danger</a>
<a href="" class="btn btn-warning">Warning</a>

<button class="btn btn-success">Success</button>
<button class="btn btn-info">Info</button>

Botones outline

Contamos con las clases btn-outlined o btn-outline para eliminar el color de fondo de los botones.

Default Primary
Danger Warning 

<a href="" class="btn btn-outlined">Default</a>
<a href="" class="btn btn-primary btn-outlined">Primary</a>

<button class="btn btn-secondary btn-outline">Secondary</button>
<button class="btn btn-link btn-outline">link</button>

<a href="" class="btn btn-danger btn-outlined">Danger</a>
<a href="" class="btn btn-warning btn-outlined">Warning</a>

<button class="btn btn-success btn-outline">Success</button>
<button class="btn btn-info btn-outline">Info</button>

Tamaños

Modifica el tamaño de los botones mediante las clases btn-lg, btn-md y btn-sm.


<button class="btn btn-primary btn-lg">Botón LG</button>
<button class="btn btn-primary btn-md">Botón MD</button>
<button class="btn btn-primary">Default</button>
<button class="btn btn-primary btn-sm">Botón SM</button>

Desactivado

Modifica el estado de un botón a desactivado mediante la clase disabled o como atributo.

Botón atributo disabled Botón clase disabled 

<a class="btn" disabled>Botón atributo disabled</a>
<a class="btn disabled">Botón clase disabled</a>

Botón redondeado

Redondea los botones usando btn-round o btn-rounded.

Botón rounded Botón round 
<a class="btn btn-success mr-md btn-rounded">Botón rounded</a>
<a class="btn btn-secondary btn-round">Botón round</a>

Botón como bloque

Usando la clase btn-block, hacemos que ocupe el ancho completo del padre.


<button class="btn btn-block btn-primary">Botón block</button>

También podemos usar esta clase para responsive, es decir, utilizando la clase btn-#{breakpoint}-block (donde #{breakpoint} equivale a lg, md, sm y xs) decidimos para que resolución el botón ocuparía el ancho completo.

Botón lg block Botón md block Botón sm block Botón xs block 

<a class="btn btn-primary btn-lg-block">Botón lg block</a>
<a class="btn btn-primary btn-md-block">Botón md block</a>
<a class="btn btn-primary btn-sm-block">Botón sm block</a>
<a class="btn btn-primary btn-xs-block">Botón xs block</a>
