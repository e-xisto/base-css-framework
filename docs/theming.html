Theming

Modifica todo el estilo de la web simplemente modificando el archivo variable.scss.

Instrucciones para cambiar variables

Para cambiar una variable debemos acceder al archivo variable.scss, donde encontraremos definidas todas las variables del framework.

Uno de los factores a tener en cuenta es que no debemos cambiar el nombre de las variables dentro de este archivo, simplemente cambiaremos su valor ya que si modificamos su nombre nos dará error al compilar el CSS.

Dicho error se debe a que, al modificar el nombre de la variable, la llamada a ella en el archivo sass que aparece definida no existe, por lo tanto devuelve un error avisando de esto. Por este motivo, nunca debemos modificar el nombre de las variables, simplemente, en caso de que lo necesitemos, añadiremos una nueva variable pero nunca modificaremos una ya predefinida.

Breakpoints

Mediante los breakpoints definimos el ancho mínimo de cada ventana, permitiendonos modificar los elementos conforme la resolución de la ventana cambia.


$media-xs: 767px;
$media-sm: 768px;
$media-md: 992px;
$media-lg: 1200px;

$breakpoints: (
    xs: $media-xs,
    sm: $media-sm,
    md: $media-md,
    lg: $media-lg
);

Instrucciones para sobreescribir componentes o elementos dentro de la carpeta theme

Por ejemplo, si queremos redefinir el componente tabs.scss lo primero que haremos será hacer una copia del componente a la carpeta /theme. A continuación, comentaremos la línea donde se realiza la importación de este componente en /components/_all.scss y añadiremos un comando @import './theme/tabs'; dentro del archivo base.scss.

Este paso lo debemos realizar tanto para sobreescribir componentes como elementos.
