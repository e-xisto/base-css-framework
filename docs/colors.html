Colores

Una amplia y modificable paleta de colores.

Estos colores se pueden modificar sencillamente en el archivo variables.scss. Tienen una semántica muy sencilla para aplicar en CSS ya que todos comienzan por $color-*

Color primario

primary-dk
primary-md
primary
primary-lt
primary-br

Color secundario

secondary-dk
secondary-md
secondary
secondary-lt
secondary-br

Colores personalizados

custom-1
custom-2
custom-3
custom-4

Colores de sistema

success
warning
danger
info

Escala de grises

black
dark
midtone
gray
light
bright
white
